# Community Health Systems (community-health-systems)
Community Health Systems is a Fortune 500 hospital operator that owns, leases, and operates general acute care hospitals across the United States. In compliance with the CMS Interoperability and Patient Access Final Rule (CMS-9115-F), Community Health Systems provides FHIR-based healthcare interoperability APIs that enable third-party applications to access patient data, adjudicated claims, encounters, formulary information, and provider directories.

## APIs

### Community Health Systems Patient Access API
Community Health Systems provides healthcare interoperability APIs pursuant to the CMS Interoperability and Patient Access Final Rule (CMS-9115-F). The Patient Access API allows third-party applications to retrieve data concerning adjudicated claims, encounters, formulary data, and clinical data using FHIR (Fast Healthcare Interoperability Resources) standards.

### Community Health Systems Provider Directory API
The Community Health Systems Provider Directory API allows third-party applications and payers to retrieve provider and pharmacy directory information in compliance with CMS interoperability requirements. The API uses FHIR standards for healthcare data exchange.
